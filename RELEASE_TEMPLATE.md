# DualTime v0.0.0

DualTime shows your **local time** in both **12-hour** and **24-hour** formats — built for ham radio operators, pilots, military personnel, and live-event crews who switch between civilian and military time.

## Install

### Android

Download **DualTime-v0.0.0-beta.apk** (standalone beta — no Metro required) from [Release Assets](https://github.com/atleyllc/dualtime/releases).

Package: `com.atleyllc.dualtime`

**Home screen widget:** Long-press the launcher → Widgets → DualTime. The widget shows both time formats and refreshes on the system schedule (every 30 minutes minimum) and while the app is open.
