# DualTime v0.1.0

Initial MVP release — local time in 12-hour and 24-hour formats, side by side.

## What's new

- Dashboard with large 12-hour and 24-hour clocks, live per-second updates, and current date
- Dark minimalist UI with Atley LLC branding
- Android home screen widget showing both time formats
- GitHub Pages web build and tag-driven APK releases

## Install

### Android

Download **DualTime-v0.1.0-beta.apk** from [Release Assets](https://github.com/atleyllc/dualtime/releases).
